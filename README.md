# claude-fable-orchestrator-workflow
Sharing the orchestrator workflow I use to maximize Fable usage and efficiently route tasks to different level models through subagents.

- /model Fable
- /effort xhigh + ultracode
- Download orchestrator-workflow.md or send your agent a direct link to it
- Tell your agent to implement the workflow
- After it has created the /agents you need to restart claude code for the agents to propogate
